# Mihna-Project

This project presents the design and implementation of a professional networking platform tailored to the Algerian market. The platform aims to offer a local alternative to global solutions such as LinkedIn by focusing on accessibility, cultural relevance, and support for local professionals, freelancers, and business owners.
The proposed solution is implemented as a web application using a modern frontend stack based on React, TypeScript, and TanStack Router for type-safe and file-based routing. On the backend, Node.js and Express.js are used to build the server-side logic and API, while MongoDB serves as the main NoSQL database for storing user data, profiles, posts, and other platform content. Custom CSS is employed to create a clean and responsive interface adapted to different devices.
The report is structured into two main chapters. The first chapter analyzes the concept and market need, describing the challenges faced by Algerian professionals and the added value of a localized professional networking platform. The second chapter details the technical implementation, including the architecture, data modeling, core features, and development choices made throughout the project.
In addition to the technical contribution, this project highlights the importance of designing digital tools that take into account local constraints, language, and culture, in order to support professional communities in emerging markets.
